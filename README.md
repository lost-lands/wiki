## Building

### Requirements
The Lost Lands Wiki requires both [MKDocs](https://www.mkdocs.org) and [Material for MkDocs](https://squidfunk.github.io/mkdocs-material) to be installed.

```sh
$ pip install mkdocs
$ pip install mkdocs-material
```

## Serving
```sh
$ mkdocs serve
```
